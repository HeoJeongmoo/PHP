🍽️ 식당 예약 시스템
1. 예약 기능

예약 정보 입력: 날짜, 시간, 식당, 인원수, 메뉴 선택
예약자 정보 입력: 이름, 전화번호, 이메일, 비밀번호
예약 확인: 입력한 정보 최종 확인
예약 완료: 데이터베이스에 저장

2. 예약 조회 기능

로그인: 이메일 + 비밀번호로 인증
예약 목록 조회: 본인이 예약한 내역 확인
영수증 출력: JPG/PNG 형태로 영수증 다운로드

3. 예약 검증 기능

중복 예약 방지: 같은 시간대 예약 불가 체크
입력값 검증: 이메일, 전화번호, 비밀번호 형식 체크

❓ Q&A 게시판
4. 질문 관리

질문 작성: 제목, 내용, 식당 선택하여 질문 등록
질문 목록: 식당별 Q&A 목록 조회
질문 상세보기: 질문 내용과 답변들 확인

5. 답변 시스템

답변 작성: 질문에 대한 답변 등록
답변 채택: 질문 작성자가 원하는 답변 채택
채택 표시: 채택된 답변 구분 표시

6. 사용자 인증

Q&A 로그인: 예약한 사용자만 Q&A 이용 가능
권한 관리: 본인이 작성한 질문만 답변 채택 가능
